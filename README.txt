Blog Engine

By Thomas Turgeon

Prerequisites:
    NodeJs : https://nodejs.org/en
    Bootstrap: npm i bootstrap@5.3.3


How to use:
    1. Open Gitbash and navigate to the directory of the project ex: .../documents/nwareProject
    2. node server.js to start the server
    3. http://localhost:3000/AdminInterface.html for the Main part of the project
    4. http://localhost:3000/posts for all the posts in the server
    5. http://localhost:3000/posts/[ID] for the post with the specified ID
    6. http://localhost:3000/categories for all the categories in the server
    7. http://localhost:3000/categories/[ID] for the category with the specified ID
    8. http://localhost:3000/categories/[ID]/posts for all the posts with the specified category ID
